This file gives a brief description of how to use this code.

SELECTOR is an algorithm for assembling an individual human mobility prediction model.

The main file is 'Main_SELECTOR.py'.

It accesses database according to the specification given in the file 'Main_SELECTOR.py'.

The corresponding database scheme is given by the file 'selector_database_scheme.sql'.

All other files are helpers or data structures supporting the execution of SELECTOR. 

The corresponding documentation and purpose of each of these files is given in each of these files.
